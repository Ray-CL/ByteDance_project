# 学习总结

## 问题 1：主线程网络请求
- 问题：在 UI 线程直接请求网络会导致 ANR。
- 解决：使用协程 + Dispatchers.IO 处理网络与文件 IO。
- 优化思路：为短暂网络异常加入重试与退避策略。

## 问题 2：重复下载图片
- 问题：应用重启后可能再次下载相同图片。
- 解决：使用磁盘缓存（filesDir/images）并先判断文件是否存在。
- 优化思路：引入 Room 记录缓存信息与淘汰策略。

## 问题 3：Bitmap 解码带来的内存压力
- 问题：大图解码导致内存峰值和滚动卡顿。
- 解决：使用 LruCache 复用已解码的 Bitmap。
- 优化思路：基于视图尺寸进行下采样解码。
