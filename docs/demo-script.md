# 演示脚本（<= 3 分钟）

1. 打开应用，展示双列图片流。
2. 下拉刷新，展示新内容。
3. 下滑触发分页加载。
4. 点击图片进入详情页。
5. 在详情页点击收藏/取消收藏。
6. 返回列表点击卡片收藏按钮。
7. 切换到“收藏”Tab 展示收藏列表。
8. 说明元数据（日期、大小、路径）。
9. 结束应用后重新打开，展示缓存命中（不会重复下载）。
10. 点击“清除缓存”，展示图片重新下载。
11. 简要讲解代码结构：MainActivity、ImageRepository、ImageAdapter。
12. 说明关键技术难点与解决方案（参考 learning-summary.md）。
